# Accessibility — About page is keyboard navigable

## Context
- The new /about page must be accessible via keyboard
- All links and interactive elements should be reachable via Tab

## Steps
1. Navigate to /about
2. Press Tab repeatedly to cycle through interactive elements

## Expected
- All links on the about page are reachable via Tab key
- Focus indicators are visible on each focused element
- The "back to home" link is keyboard accessible
