# Visual-design — Content has a max-width for readability

## Context
- Browser window is wider than 1200px

## Steps
1. Navigate to the homepage on a wide monitor

## Expected
- Page content does not stretch to full viewport width
- Content is constrained to a readable max-width (roughly 900-1200px)
- Content is horizontally centered within the viewport
