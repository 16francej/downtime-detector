# Visual-design — Clear typography hierarchy

## Context
- App is loaded at the root URL

## Steps
1. Navigate to the homepage
2. Observe the page title, section headings, body text, and data labels

## Expected
- Page title is the largest text element and uses a bold weight
- Section headings (chart heading, table heading) are visually distinct from body text
- Data labels and metadata use a smaller, lighter treatment
- Font family is consistent (Geist sans-serif) across all text elements
- No generic system fonts (Arial, Helvetica) visible
